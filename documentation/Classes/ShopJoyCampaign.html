contentType

The content type of the campaign:

“html”, that can be parsed in a webview
“push”, for a push message only
“text”, plain text that can be used for anything
“url”, a URL that can be presented in a webview or browser
“facebook”, a Facebook ID that can be shown with a URL scheme, webview or browser
“twitter”, a Twitter ID that can be shown with a URL scheme, webview or browser
“instagram”, a Instagram ID that can be shown with a URL scheme, webview or browser
“directsmile”, a DirectSmile URL that can be presented in webview or browser

@property NSString *contentType
